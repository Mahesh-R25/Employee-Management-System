# Employee Management System

## 📌 Description
This is a console-based Employee Management System developed using Core Java.  
The application allows users to manage employee records efficiently using CRUD operations.

## 🚀 Features
- Add Employee
- View Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee
- Sort Employees by Salary
- Persistent Data Storage using File Handling (Serialization)

## 🛠️ Technologies Used
- Java (Core Java)
- OOP Concepts
- Collections (ArrayList)
- File Handling (ObjectInputStream / ObjectOutputStream)

## 📂 Project Structure
Employee-Management-System
 └── EmployeeManagement.java

## ▶️ How to Run
1. Open the project in IntelliJ IDEA or any Java IDE
2. Compile and run `EmployeeManagement.java`
3. Use the menu options to interact with the system

## 🎯 Purpose
This project demonstrates strong understanding of Java fundamentals, object-oriented programming, and file handling.
